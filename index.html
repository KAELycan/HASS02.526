Pollutants Standards Index (PSI) Data

Air Quality per PSI Value:

Good (0-50)

Moderate (51-100)

Unhealthy (101-200)

Very Unhealthy (201-300)

Hazardous (Above 300)

For each pollutant, a sub-index is calculated from a segmented linear function that transforms ambient concentrations onto a scale extending from 0 through 500.
